Self-Host

Your data, your rules

Raven ships as a single Docker Compose file. Run it on a Raspberry Pi, a VPS, or your on-premise Kubernetes cluster. No external dependencies, no data leaves your network.

One command setup

Copy .env.example, run docker compose up and you're live.

Runs on edge hardware

Optimised for Raspberry Pi 5 and low-memory ARM nodes. Native Go binary, no JVM overhead.

Automatic HTTPS

Traefik with Let's Encrypt or Cloudflare Tunnel — zero-port-exposure ingress included.

Read the Quickstart Guide
terminal

# Clone and configure

$ git clone https://github.com/ravencloak-org/Raven

$ cp .env.example .env

$ nano .env # set your secrets


# Start everything

$ docker compose up -d


# ✅ Raven is running at http://localhost:8080

# ✅ Keycloak at http://localhost:8180

# ✅ OpenObserve at http://localhost:5080
